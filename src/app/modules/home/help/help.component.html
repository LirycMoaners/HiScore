Navigate on HiScore app

When you use the app, you always have the possibility to access to another page with the main menu

There is 4 entries on the main menu :
    - Home: where you can see all your games
    - New game: to start a new game
    - Help: where you can find how to use the app (you are here)
    - About: all the informations about the app, the developer and legal terms

Start a new game

On the home page, click on the "+ Start a new game" button to create a new game, or click on "New game" from the main menu.

The game creation page appears. On this page, you can parameter :
    - Players by entering their name
    - Which game you're going to play
    - If the end of the game is decided by the number of rounds or a player score
    - How many rounds or what score a player should make to end the game
    - If the best player should be the one with the lowest or the highest score
    - If the first player to start should be choose randomly

Click on "Start game" and you will arrived on the current game page.

Writing score

There's many ways to write score for each player.
If you click on the round score, you can edit it directly with your keyboard.
If you click on "+" or "-", it will add or remove 1 point to the round score.
If you let clicked "+" or "-", the score dialog should appear. On this dialog, you can click on the round score to edit it directly or click on the different buttons to add or remove points to the round score.

If you want to change a player round score, you can just click on this particular score on the right part of his tile. The score dialog will show up and you will be able to edit his score.

Game menu options

If you open the game menu from the main bar during a game, there will 4 entries :
    - New game: to start a new game with same players and parameters
    - Modify game: to change players or parameters on the current game
    - Edit last round: to come back to the last round and change scores
    - End game: to say that the game is end and it will show the ranking
